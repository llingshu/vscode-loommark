# LoomMark 富渲染增强设计（2026-07-17）

## 目标

在现有 CodeMirror 6 源码保留式编辑器上，补齐图片预览、GFM 表格、任务列表/列表、
引用块等富渲染能力，整体交互向 Typora 式"预览切换"体验靠拢：光标在块外显示渲染
结果，光标进入展开为 Markdown 源码编辑。

## 交互模型（已确认）

**预览切换式**：所有新增富渲染均遵循既有标题/粗体的隐藏逻辑——
光标不在语法范围内时渲染，进入时还原源码。不做原地富编辑（表格单元格
内嵌编辑器等），见"不做的"。

## 前置步骤

当前工作区约 520 行未提交的代码块增强（mac 风格工具栏、行号、
`@codemirror/language-data` 语法高亮）测试全绿，先单独提交，再开始本设计的工作。

## 1. 共享基础设施（重构）

`webview/main.ts`（809 行）拆分为三个模块：

| 模块 | 职责 | 依赖 |
| --- | --- | --- |
| `webview/markdown-ranges.ts` | 纯函数解析：fenced/inline code（移入）、表格块、图片、任务项、列表项、引用块、水平线 | 无 DOM，可被 node 测试 |
| `webview/widgets.ts` | 全部 `WidgetType`：代码工具栏（移入）、表格、图片、复选框 | DOM + markdown-ranges |
| `webview/main.ts` | 编辑器装配、同步协议、大纲、消息处理 | 上述两者 |

约束：块级替换装饰（表格、独占行图片）必须由 `StateField` 提供
（CodeMirror 限制，ViewPlugin 不可提供 block decoration）。沿用
`codeToolbarField` 模式，在 `docChanged || selection` 时重建。

## 2. 图片预览

- 语法 `![alt](src)`；独占一行的图片用 block widget 替换整行，行内混排用
  inline widget 替换该语法范围。
- 路径解析：`init` 消息中已有 `resourceBase`（`extension.ts:173`）存入 webview
  状态；相对路径拼 `resourceBase`，`http(s)`/`data:` 原样使用（CSP 已放行）。
- 失败占位符：图标 + alt + 路径文本。
- 尺寸：`max-width: 100%; max-height: 480px`。
- 点击图片 → 光标定位到源码位置，展开编辑。

## 3. GFM 表格

- 识别：表头行 + `|---|` 分隔行 + 后续数据行组成的连续行块。
- 整块替换为 `<table>` widget：
  - 单元格内轻量渲染粗体/斜体/行内代码/链接（复用现有 inline 正则）；
  - 对齐遵循 `:---:` / `---:` / `:---`。
- 交互模式由 `loommark.table` 设置控制（2026-07-18 变更，默认从源码展开改为富编辑）：
  - `rich`（默认）：表格始终渲染；点击单元格进入原地编辑，编辑区显示该单元格的
    **原始源码**；Enter/失焦提交（一次最小 `dispatch` 替换单元格源码范围，`|` 自动
    转义，可撤销），Esc 取消，Tab/Shift+Tab 提交并移动到下/上一个单元格。
    选择提交制而非逐键同步是为了 IME 组合输入安全与光标稳定。
  - `source`：光标在块外渲染、进入展开为源码（原设计行为）。
- 配置变更通过 `StateEffect` 触发装饰重建，即时生效。

## 4. 任务列表、列表、引用块、水平线

- `- [ ]` / `- [x]`：标记替换为复选框 widget；**点击复选框直接切换源码中的
  `x`，无需进入源码**；光标在该行时显示源码。
- 无序列表 `-` `*` `+`：光标在外时标记按嵌套层级渲染为 `•` `◦` `▪`；
  有序列表数字保留原样。
- 引用块 `>`：整行加左侧竖线样式；光标在外时隐藏 `>` 标记。
- 水平线 `---` / `***` / `___`：光标在外时渲染为分隔线。

## 5. 顺带修正

- 代码块深色底为刻意的"终端窗口"风格，保留；浅色主题下检查工具栏与代码体的
  边框衔接，微调使其不突兀。
- 新增解析函数全部配 node 测试：`test/markdown-ranges.test.mjs`。

## 6. 不做的（YAGNI）

- 表格加行/加列工具栏（单元格原地编辑已于 2026-07-18 实现，见第 3 节）。
- 图片缩放手柄、粘贴上传图片。
- 脚注。
- 数学公式原列于此；2026-07-19 起已实现：KaTeX 本地渲染，行内 `$...$` 与块级
  `$$...$$`（含跨行），货币启发式防误判，交互沿用预览切换式。字体经 esbuild
  以 woff2 data URI 内嵌，woff/ttf 回退用 `empty` loader 裁剪。

## 错误处理

- 图片加载失败：占位符，不抛错。
- 表格解析失败或格式残缺：不渲染 widget，保持源码显示（安全回退）。
- 所有 widget `contentEditable = false`，点击行为通过 `view.dispatch` 定位
  光标，不直接改 DOM。

## 测试

- 交互（复选框点击、表格展开）：Extension Development Host 手动验证。
