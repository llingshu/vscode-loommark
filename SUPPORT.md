# Support

Use [GitHub Discussions](https://github.com/llingshu/vscode-loommark/discussions) for usage questions,
configuration help, and ideas. Use the issue templates for reproducible defects and feature
requests.

Before asking for help, update to the latest release, review the README settings and limitations,
and test with unrelated extensions temporarily disabled. Include the LoomMark version, VS Code
version, operating system, relevant settings, and a minimal Markdown sample when possible.

Security reports must follow [SECURITY.md](SECURITY.md), not public support channels.
